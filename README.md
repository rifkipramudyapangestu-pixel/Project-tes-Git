# Project-tes-Git
1. Git (Version Control System)
Git adalah sistem kontrol versi terdistribusi (Distributed Version Control System/DVCS) yang diciptakan oleh Linus Torvalds pada tahun 2005. Sebagai DVCS, setiap pengembang memiliki salinan lengkap dari seluruh riwayat repositori, yang memungkinkan operasi seperti commit dan branching dilakukan secara lokal tanpa koneksi internet. Git melacak perubahan pada konten file, bukan pada perbedaan antar file, yang membuatnya sangat efisien dan cepat.

2. GitHub (Remote Repository Hosting)
GitHub adalah platform berbasis web yang menyediakan layanan hosting repositori Git. GitHub memfasilitasi kolaborasi tim dengan menyediakan antarmuka grafis untuk mengelola repositori, melacak issue, meninjau kode (code review), dan mengelola permintaan penggabungan (Pull Request). GitHub bertindak sebagai repositori sentral (remote) tempat semua anggota tim dapat menyinkronkan pekerjaan mereka.
